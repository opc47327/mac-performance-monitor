## Sensei 是什么？

Sensei 是由瑞典开发商 Cindori（就是做 Trim Enabler 那家）出品的 Mac 系统工具，十多年的 Mac 系统研究积累，做出来的东西确实有点东西。

它不是那种"一键清垃圾"就完事的玩意儿。它更像是把活动监视器、磁盘工具、系统偏好设置里散落的各种东西，融合成一个统一的控制台——界面好看，操作直觉，什么都看得到。

核心功能大概这几块：

- **菜单栏实时监控**：CPU、GPU、内存、风扇转速、温度，全部在菜单栏里一眼扫完，不用开 app
- **高级仪表盘**：统一查看电池健康度、硬盘占用、设备型号等软硬件信息
- **磁盘清理**：扫描缓存、日志、下载残留，一键清理，不碰个人文件
- **彻底卸载应用**：把 app 拖进废纸篓只是删了图标，Sensei 的卸载器能把所有残留一起清掉
- **登录项管理**：关掉那些你根本不知道在后台跑的进程，开机快、运行也顺
- **硬件健康监控**：电池循环次数、S.M.A.R.T. 磁盘状态、散热传感器完整视图
